# GPUI Unofficial Docs

## Dependencies

- [mdbook](https://rust-lang.github.io/mdBook/guide/installation.html)

#### Install mdbook

```bash
cargo install mdbook
```

## Installation

1. Clone the repository.

```bash
git clone https://github.com/Himasnhu-AT/GPUI-docs-unofficial.git
```

2. Build book:

```bash
mdbook build
```

3. View Docs:

```bash
mdbook serve
```
